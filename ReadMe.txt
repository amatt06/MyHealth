MyHealth Application

Java Version: JDK 20

Dependencies:
- JavaFX
- SQLite JDBC Driver 

IDE/Compiler:
- IntelliJ IDEA (version 17.0.6)

Instructions to Compile, Test, and Run the Program:
1. download the source code.
2. Open the project in IntelliJ IDEA or your preferred IDE.
3. Configure the project's JDK to use JDK 20.
4. Resolve and download any missing dependencies.
5. Build the project.
6. Run the program by executing the main class (launch() from MyHealthApp) or launching the application from the IDE.
